After Dark

The admin tool Canvas forgot to build.

A Chrome extension for Canvas LMS administrators. Faster course search, cross-environment comparison, course watchlisting, org visualization, people tools, and more — all without leaving your browser.

Get Started View on GitHub

What After Dark Does

Everything you've been doing in five browser tabs, done in one.

🔍
Course Search
Search by name, instructor, or section number. Filter by term, sub-account, publish status, enrollment. Browse thousands of courses in seconds.
👤
People Search
Find any user by name or login. See enrollments grouped by term, page view history, last login status, and profile details.
⚖️
Course Comparison
Select two courses and compare their content side by side. See what's in one but not the other, with diff badges on every section.
🌐
Cross-Environment Compare
Compare the same course across Live, Test, and Beta environments simultaneously. See exactly what differs and which version is newer.
💀
The Vault
Watchlist for courses you need to come back to. Add notes, archive old entries, and access your saved courses instantly.
🔗
Similar Activity
Compare two users' page view history and highlight matching activity within a configurable time window. An activity timeline makes patterns visible at a glance.
📊
Org Visualization
Sunburst and tree charts of your sub-account structure with live course counts. See your institution's hierarchy at a glance.
🔬
Course Audit
Deep-dive into a course's grading patterns, discussion activity, Kaltura usage, and content inventory. Flag potential academic integrity concerns.
🗺️
Blueprint Tools
Browse master courses and their associated children. View sync history and compare blueprint children side by side.
🎭
Demo Mode
Scramble names, SIS IDs, and email addresses for safe screen sharing in meetings or presentations.
🎨
Themes
17 colour themes including every UW System institution. Switch instantly — your choice persists across sessions.
🏛️
Multi-Institution
Connect multiple Canvas environments. Switch between institutions with one click from the home screen.
💡 After Dark is built for Canvas administrators. It uses your existing browser session — no API token storage, no external servers, no third-party data sharing.
