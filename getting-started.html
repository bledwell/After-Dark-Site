Getting Started

After Dark is a Chrome extension for Canvas LMS administrators. Install it once and it works automatically alongside any Canvas tab you have open.

After Dark is free and available on the Chrome Web Store.

Download Chrome Extension

Step 1 — Install the Extension

After Dark is distributed through the Chrome Web Store. It works in Chrome and any Chromium-based browser (Edge, Brave, Arc).

  1. Click the Install from Chrome Web Store button above.
  2. Click Add to Chrome on the store listing.
  3. Confirm by clicking Add extension in the dialog.
  4. The After Dark icon will appear in your browser toolbar.
Tip: If you don't see the icon right away, click the puzzle piece icon in your toolbar and pin After Dark so it's always visible.

Step 2 — Run the Setup Wizard

The first time you open After Dark, a setup wizard walks you through connecting to your Canvas instance. This only needs to be done once.

Setup Wizard walkthrough — connecting After Dark to your Canvas instance.
  1. Click the After Dark icon in your toolbar to open it.
  2. The setup wizard launches automatically on first run.
  3. Enter your institution name.
  4. Enter your Canvas URL (e.g. uwosh.instructure.com). After Dark will attempt to detect this automatically if you have a Canvas tab open.
  5. Optionally enter a vanity URL if your institution uses one (e.g. canvas.uwosh.edu).
  6. Click Finish. After Dark will verify the connection and take you to the main search screen.
Connecting to Canvas: After Dark works by routing requests through your existing Canvas browser session — no API token or separate login required. Just make sure you have a Canvas tab open and logged in before opening After Dark.
Cross-environment users: If you need to compare courses across Live, Test, and Beta environments, you can add a second Canvas URL in Settings after the initial setup. Open both environment tabs in your browser before running comparisons.
